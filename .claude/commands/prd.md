# /prd
You will produce a one-page PRD for the task I describe.

Output format (must follow exactly):
1) Goal (1 sentence)
2) Non-Goals (bullets)
3) Deliverables (bullets)
4) Acceptance Criteria (bullets, testable)
5) Constraints (Windows/PowerShell first)
6) Plan (<= 6 lines)
7) Risks / Open Questions (bullets)

Rules:
- If the task is ambiguous, ask at most 1 precise question OR state 2 assumptions.
- Keep it concise and actionable.
